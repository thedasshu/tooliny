## Installation & Usage

### 1. Install basic package for further installation
```bash
apt update && apt upgrade -y
apt install git -y
```

### 2. Clone the Repo:
```bash
git clone https://github.com/thedasshu/tool-installer
cd tool-installer
bash install. sh
```
